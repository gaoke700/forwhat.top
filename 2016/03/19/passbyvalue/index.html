传值与传址

2016-03-19
传值 传址 克隆
  1. 传值
  2. 传址
  3. 克隆

传值

传值:把一个变量的值复制一份给另一个变量

  1. 复制变量的值
  2. 申请一个新的内存地址存放复制的值
  3. 把复制的值的地址告诉另一个变量
    简单数据类型的赋值是传值

例:

var a = 1;
var b = a;
b = 2;
console.log(a,b)                //输出1和2

传址

传址：把一个变量在内存中的地址复制给另一个变量.

复杂数据类型的赋值是传址。

例:

var arr1 = [1,2,3,4];
var arr2 = arr1;
arr2[0] = 10;
console.log( arr1, arr2 );        //两个输出的都是 [10, 2, 3, 4]

通过比较我们发现传址会改变原变量的值,而传值不会.

克隆

要想传址时还不改变原变量的值,我来给介绍几种方法:

1.通过循环把传址改为传值:

var arr1=[1,2,3,4]
var arr2=[]
for(var i=0;i<arr1.length;i++){
    arr2[i]=arr1[i]
}

arr2[0] = 10;
console.log( arr1, arr2 );            //分别输出[1,2,3,4]和[10,2,3,4]
